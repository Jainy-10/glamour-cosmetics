# 🌟 Glamour Cosmetics & Jewelry Website

## Project Summary
A complete e-commerce website for cosmetics and jewelry sales, featuring modern design, admin panel, and responsive layout.

## Key Features
✅ Responsive design (mobile, tablet, desktop)
✅ Admin authentication system
✅ Product management (CRUD operations)
✅ Shopping cart functionality
✅ Contact form with validation
✅ Modern UI with animations
✅ Secure PHP backend
✅ MySQL database integration

## Technology Stack
- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
- **Backend:** PHP 7.4+, MySQL
- **Libraries:** Font Awesome, jQuery
- **Development:** XAMPP local server
