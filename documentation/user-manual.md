# 📖 User Manual

## For Website Visitors

### Homepage
- View featured products and promotions
- Navigate to different sections
- Learn about the brand

### Products Page
- Browse all available products
- Use search functionality
- Filter by categories
- View product details

### Shopping Cart
- Add products to cart
- Update quantities
- Remove items
- View total price

### Contact Us
- Send inquiries via contact form
- View store location and hours
- Access FAQ section

## For Administrators

### Login Process
1. Go to `/admin/login.php`
2. Enter credentials (admin/admin123)
3. Access admin dashboard

### Product Management
1. **Add Products:** Use "Add Product" form
2. **Edit Products:** Click edit icon on products list
3. **Delete Products:** Use delete button (with confirmation)
4. **Upload Images:** Select and upload product images

### Dashboard Features
- View total products count
- Monitor recent activities
- Quick access to all admin functions
